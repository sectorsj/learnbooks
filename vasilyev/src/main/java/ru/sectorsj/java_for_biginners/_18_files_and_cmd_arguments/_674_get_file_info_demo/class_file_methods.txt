01 |canExecute()            |      Результатом метод возвращает логическое значение,                  |
   |                        |      определяющее, является ли файл исполняемым                         |
   |                        |      (может ли он быть запущен на выполнение)                           |
---|------------------------------------------------------------------------------------------------- |
02 |canRead()               |      Результатом метод возвращает логическое значение,                  |
   |                        |      определяющее, доступен ли файл для чтения                          |
---|------------------------------------------------------------------------------------------------- |
03 |canWrite()              |      Результатом метод возвращает логическое значение,                  |
   |                        |      определяющее, доступен ли файл для записи                          |
---|------------------------------------------------------------------------------------------------- |
04 |createNewFile()         |      Методом создается новый пустой файл с названием,                   |
   |                        |      определяющим на основе объекта, из которого вызывается             |
   |                        |      метод. Файл создается только при условии, что файла с              |
   |                        |      данным именем еще не существует                                    |
---|------------------------------------------------------------------------------------------------- |
05 |delete()                |      Метод используется для удаления файла, соответствующего            |
   |                               объекту, из которого вызывается метод                              |
---|------------------------------------------------------------------------------------------------- |
06 |deleteOnExit()          |      Метод предназначен для удавления файла (определяемого              |
   |                        |      объектом, из которого вызывается метод) перед завершением          |
   |                        |      работы виртуальной машины                                          |
---|------------------------------------------------------------------------------------------------- |
07 |exists()                |      Метод позволяет проверить существует ли файл                       |
---|------------------------------------------------------------------------------------------------- |
08 |getAbsoluteFile()       |      Методом возвращается объект класса File, соответствущий            |
   |                        |      абсолютному пути для объекта, из которого вызывается метод         |
---|------------------------------------------------------------------------------------------------- |
09 |getAbsolutePath()       |      Методом возвращается текст, соответствующий абсолютному            |
   |                        |      пути объекта, из которого вызывается метод                         |
---|------------------------------------------------------------------------------------------------- |
10 |getCanonicalFile()      |      Методом возвращается объект класса File, соответствующий           |
   |                        |      каноническому пути для объекта, из которого вызывается метод       |
---|------------------------------------------------------------------------------------------------- |
11 |getCanonicalPath()      |      Методом возвращается текст, соответствующий каноническому          |
   |                        |      пути для объекта, из которого вызывается метод                     |
---|------------------------------------------------------------------------------------------------- |
12 |getName()               |      Методом возвращается текстовое значение с названием файла          |
---|------------------------------------------------------------------------------------------------- |
13 |getParent()             |      Методом возвращается название родительской директории              |
---|------------------------------------------------------------------------------------------------- |
14 |getParentFile()         |      Методом возвращается объект класса File, соответствующий           |
   |                        |      родительской директории                                            |
---|------------------------------------------------------------------------------------------------- |
15 |getPath()               |      Методом возвращается тестовое значение с указанием полного         |
   |                        |      пути к файлу                                                       |
---|------------------------------------------------------------------------------------------------- |
16 |isAbsolute()            |      Методом выполняется проверка, содержит ли объект, из               |
   |                        |      которого вызывается метод, абсолютный путь к файлу                 |
---|------------------------------------------------------------------------------------------------- |
17 |isDirectory()           |      Методом выполняется проверка, соответствует ли, объект             |
   |                        |      из которого вызывается метод, директории                           |
---|------------------------------------------------------------------------------------------------- |
18 |isFile()                |      Методом выполняется проверка, соответствует ли объект, из          |
   |                        |      которого вызывается метод, файлу                                   |
---|------------------------------------------------------------------------------------------------- |
19 |isHidden()              |      Метод позволяет проверить, является ли файл скрытым                |
---|------------------------------------------------------------------------------------------------- |
20 |lastModified()          |      Метод возвращает время (в миллисекундах) последнего изменения      |
   |                        |      файла                                                              |
---|------------------------------------------------------------------------------------------------- |
21 |list()                  |      Методом возвращается массив с названиями файлов и директорий,      |
   |                        |      находящихся в директории, определяемой объектом, из которого       |
   |                        |      вызывается метод                                                   |
---|------------------------------------------------------------------------------------------------- |
22 |listFiles()             |      Методом возвращается массив объектов класса File,                  |
   |                        |      соответствущих файлам, находящимся в директории, определяемой      |
   |                        |      объектом, из которого вызывается метод                             |
---|------------------------------------------------------------------------------------------------- |
23 |listRoots()             |      Статический метод возвращает результатом массив объектов           |
   |                        |      класса File, определяющих корневые директории системы              |
---|------------------------------------------------------------------------------------------------- |
24 |mkdir()                 |      Методом создается директория, соответствующая объекту, из          |
   |                        |      которого вызывается метод                                          |
---|------------------------------------------------------------------------------------------------- |
25 |mkdirs()                |      Методом создается директория,определяемая объектом, из             |
   |                        |      которого вызывается метод. Если при этом адрес конечной            |
   |                        |      директории содержит несуществующие внешние директории, они         |
   |                        |      тоже создаются                                                     |
---|------------------------------------------------------------------------------------------------- |
26 |renameTo()              |      Метод позволяет переименовать файл, определяемый объектом, из      |
   |                        |      которого вызывается метод. Аргумент метода - объект, класса        |
   |                        |      File, определяющий конечный файл                                   |
---|------------------------------------------------------------------------------------------------- |
27 |setExecutable()         |      Методом устанавливается или отменяется (в зависимости от           |
   |                        |      значения логического аргумента) разрешение на выполнение файла     |
---|------------------------------------------------------------------------------------------------- |
28 |setLastModified()       |      Методом определяется время последнего изменения файла              |
---|------------------------------------------------------------------------------------------------- |
29 |setReadable()           |      Методом устанавливается или отменяется (в зависимости от           |
   |                        |      значения логического аргумента) разрешение на чтение файла         |
---|------------------------------------------------------------------------------------------------- |
30 |setReadOnly()           |      Метод задает для файла или директории режим, разрешающий           |
   |                        |      только чтение                                                      |
---|------------------------------------------------------------------------------------------------- |
31 |setWritable()           |      Методом устанавливается или отменяется (в зависимости от           |
   |                        |      значения логического аргумента) разрешение на запись в файл        |
---|------------------------------------------------------------------------------------------------- |
32 |toString()              |      Метод для преобразования объекта класса File в текстовое           |
   |                        |      значение                                                           |
---|--------------------------------------------------------------------------------------------------|
